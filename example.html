Dependencies

Sparkhit is written in Java and built on top of the Apache Spark. To use Sparkhit, make sure you have a Java-1.7 or higher version installed in your operating system, as well as Spark framework. Since Sparkhit runs on a distributed computer cluster, Spark cluster should also be properly configured when running Sparkhit on cluster mode. If you want to parallelize your own tools with Sparkhit, make sure they are available on all computer nodes (eg. copy your script to each node or place it in a shared file system). Spark installation

There are different ways to install Spark on your computer. Please go to Spark`s Download page and choose a download method of your preference. If you would like to build Spark from source, please go to Building Spark page.

A simple way is to download Spark pre-build package (version 2.0.0) using command:
$ wget http://archive.apache.org/dist/spark/spark-2.0.0/spark-2.0.0-bin-hadoop2.6.tgz

Once download, untar the package:
$ tar zxvf spark-2.0.0-bin-hadoop2.6.tgz

Go to unpacked Spark directory. Under bin folder, you should see an executable spark-submit. Set environmnent variable (ENV) SPARK_HOME to your unpacked Spark directory:
$ cd ./spark-2.0.0-bin-hadoop2.6
$ pwd  Then, copy listed full path.
$ vi /home/ec2-user/.bash_profile or $ vi /home/ec2-user/.profile
Here, ec2-user refers to your user name. Paste your Spark full path in the file and set as ENV SPARK_HOME 

# bash_profile
export SPARK_HOME="/vol/ec2-user/spark-2.0.0-bin-hadoop2.6"

Reload changes in the file:
$ source /home/ec2-user/.bash_profile

Now Sparkhit should be able to recognize Spark.

☕  Notes
Sparkhit uses spark-submit executable to submit Sparkhit`s jar file to Spark cluster. As long as Sparkhit could found the full path of spark-submit, Sparkhit will work properly.

Sparkhit installation

Download Sparkhit package from github zipball:
$ wget https://github.com/rhinempi/sparkhit/zipball/master -o ./sparkhit.zip
Or tarball: $ wget https://github.com/rhinempi/sparkhit/tarball/master -o ./sparkhit.tar
Or press the download buttom on top.

Once download, unzip or untar the package:
$ tar xf ./sparkhit.tar
$ unzip ./sparkhit.zip
